## 🎨 Customization

### Adding New Habits

1. Access the dashboard
2. Click "Add New Habit"
3. Configure:
   - Name
   - Frequency (daily/weekly)
   - Times per day
   - Optional: description

### Achievement System

Current achievements include:
- 🔥 Streak milestones (7, 30, 100 days)
- 💯 Completion milestones
- ⭐ Perfect week completions
